# Elevate Labs Internship - Task 5
## Decision Trees & Random Forests

### Objective
To understand and implement tree-based machine learning models for classification.

---

## What I Did

### 1. Decision Tree Classifier
- Trained a Decision Tree Classifier using the Breast Cancer dataset.
- Evaluated model accuracy.
- Visualized the decision tree using Matplotlib.

### 2. Overfitting Analysis
- Tested the model with different tree depths.
- Compared training and testing accuracy.
- Observed overfitting at higher tree depths.

### 3. Random Forest Classifier
- Trained a Random Forest model with 100 trees.
- Compared its performance with the Decision Tree model.

### 4. Cross Validation
- Applied 5-Fold Cross Validation.
- Evaluated model stability and generalization performance.

### 5. Feature Importance
- Extracted feature importance scores from the Random Forest model.
- Visualized the most important features contributing to predictions.

---

## Results Summary

| Model | Accuracy |
|---------|----------|
| Decision Tree | ~90%+ |
| Random Forest | ~95%+ |

### Key Observation
- Random Forest achieved better accuracy than the Decision Tree.
- Random Forest showed more stable performance and reduced overfitting.
